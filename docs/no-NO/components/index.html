Komponenter

NFive is modular by design and comes in multiple components:

NFive (Påkrevd)

The NFive runtime is a FiveM server resource which handles loading the installed plugins, database management, communication and events.

MySQL Database (Påkrevd)

NFive makes use of a MySQL database for storing information and game state. See Database Setup for details.

Plugin Manager (Anbefalt)

The NFive Plugin Manager (nfpm) is a standalone tool for NFive server administration and management as well as plugin development. See Plugin Manager for details.
