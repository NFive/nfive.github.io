Solução de problemas

Erros do Servidor

Avisos esperados

Os avisos abaixo são esperados e seguros para serem ignorados:

  • INFO: No channel links found in configuration file.
  • Couldn't find resource sessionmanager.

Banco de Dados

  • mysql Error: -1 : Unable to connect to any of the specified MySQL hosts.

    Verifique o arquivoresources\nfive\config\database.yml e observe a configuração do banco de dados MySQL e teste se o banco de dados está rodando e está disponível para acesso.

  • MySql.Data.MySqlClient.MySqlPacket.ReadString ou parecido com "MySQL string error"

    Verifique se os caracteres estão configurados corretamente para NFive. Veja Configurando Banco de dados para obter instruções.

Chave de licença do FiveM

  • GlobalError: Could not authenticate server license key.

    Esse erro significa que você não tem um Chave de licença FiveM em server.cfg, verifique se o arquivo contém uma licença em sv_licensekey.

Erros do Cliente

  • System.Runtime.Serialization.dll is not a platform image (even though the dir matches).

    Esse aviso pode ser ignorado.
