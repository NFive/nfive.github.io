Resumo

Este projeto está em seu desenvolvimento inicial e ainda não é estável - essa documentação está sujeita a alterações

NFive é um gerenciador da plataforma .NET para FiveM construída para ajudar na administração de servidores e facilitar o desenvolvimento provendo plugins modulares.

Conceito

NFive foi desenvolvido para simplificar a complexa forma de gerenciar servidores e prover uma experiencia de desenvolvimento moderna aos usuários.

Como os usuários a cada dia têm demandado mais e mais detalhes nos servidores os sistemas de mod se tornam enormes, tendo assim a necessidade de uma gestão mais fácil e unificada.

Aplicação

NFive não oferece nenhuma funcionalidade dentro do jogo (isso fica por sua conta), NFive é puramente um sistema de gerenciamento e plataforma para desenvolvimento de plugins. Isso permite que o NFive permaneça agnóstico e utilizável com qualquer tipo de mod ou modo de jogo, com a funcionalidade adicionada via Plugins NFive.
