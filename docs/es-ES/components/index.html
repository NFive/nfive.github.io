Componentes

NFive es modular por diseño y viene en múltiples componentes:

NFive (Requerido)

NFive es un recurso del servidor FiveM que se encarga de cargar los complementos instalados, la administración de la base de datos, comunicación y eventos.

Base de datos MySQL (Requerida)

NFive hace uso de una base de datos MySQL para almacenar información y estado del juego. Ver Configuración de base de datos para más detalles.

Gestor de Plugins (recomendado)

El gestor de plugins NFive (nfpm) es una herramienta independiente para administración y gestión de servidores NFive, así como desarrollo de plugins. Ver Plugin Manager para más detalles.
