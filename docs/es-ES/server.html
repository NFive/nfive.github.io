Configurar servidor

NFive tiene como objetivo ser muy fácil de Iniciar y configurar, para que pueda obtener rápidamente un servidor en funcionamiento.

Configuración rápida

La forma recomendada y más fácil de instalar rápidamente un servidor FiveM y configurar NFive es usar la aplicación: NFive Plugin Manager (nfpm):

  1. Descargar la última versión de nfpm: https://dl.nfive.io/nfpm.exe

    nfpm.exe no necesita ser instalado y puede ejecutarse desde cualquier lugar.

  2. Coloque lo en la ubicación que desea instalar el servidor y abra una ventana terminal/consola en esa ubicación.

    nfpm funcionará exactamente igual en Linux/macOS si se usa con Mono: mono nfpm.exe

  3. Ejecute el comando de configuración nfpm setup y responda a las preguntas, incluyendo cómo conectarse a su base de datos.

    nfpm descargará el servidor FiveM correcto para Windows o Linux dependiendo de su plataforma.

Ahora tienes una configuración completa del servidor y puedes saltar a Verificando la instalación a continuación.

Configuración manual

Para instalar manualmente NFive en un servidor FiveM, descargue la última versión NFive y extraiga el archivo en la carpeta resources de su servidornfive/.

Como todo Resource precisa ser iniciado desde el archivo de configuración, Configuración del Servidor Para iniciarlo en la consola escribe: start nfive

NFive reemplaza completamente todos los recursos de servidor de FiveM; asegúrese de eliminarlos de su configuración si ha instalado alguno. El servidor FiveM siempre intentará cargar un recurso llamado sessionmanager, incluso si no está en la configuración de su servidor, así que debe eliminar o renombrar la carpeta de recursos si está presente.

Configuración

Los archivos de configuración NFive están ubicados en nfive/config/ con el archivo de configuración principal nfive.yml.

Conexión a la base de datos

NFive requiere una base de datos MySQL con los datos de conexión almacenados en nfive/config/database.yml:

conexión:
  host: localhost
  port: 3306
  database: fivem
  user: root
  password: password
  charset: utf8mb4
  logging: false

Verificando instalación

El directorio de su servidor debe contener un directorio `server/code> con una estructura y archivos de clave similares a:

server/
├── FXServer[.exe]
|── server.cfg
└── resources/
    └── nfive/
        ├── __resource.lua
        ├── nfive.yml
        ├── nfive.lock
        └── config/
            ├── nfive.yml
            └── database.yml
`

El Servidor FiveM en Linux está más profundamente anidado con los archivos del servidor ubicados en server/alpine/opt/cfx-server/.

Siguientes pasos

Después de ejecutar el comando de configuración tendrá un servidor FiveM en blanco con NFive instalado, ahora puede iniciarse sin ningún plugin, el servidor será inutilizable para los jugadores.

Ver la siguiente página Instalando Plugins para los plugins básicos necesarios para usar su servidor.
