Komponenty

NFive je modulární podle návrhu a má více komponentů:

NFive (Požadované)

NFive runtime je zdroj serveru FiveM který se zabývá načítáním nainstalovaných pluginů, správy databází, komunikace a událostí.

MySQL Databáze (Požadované)

NFive používá databáze MySQL pro ukládání informací a stav her. See Database Setup for details.

Plugin Manager (Doporučeno)

NFive Plugin Manager (nfpm) je samostatný nástroj pro NFive a také pro vývoj pluginu. See Plugin Manager for details.
