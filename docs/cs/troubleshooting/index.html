Řešení Problémů

Očekávaná varování

Očekává se, že následující varování na serveru FiveM budou ignorovat:

  • Couldn't find resource sessionmanager.

Chyby serverů

Database

  • mysql Error: -1 : Unable to connect to any of the specified MySQL hosts.

    Check your nfive\config\database.yml configuration is correct and that a MySQL database is running and available.

  • MySql.Data.MySqlClient.MySqlPacket.ReadString nebo podobná MySQL chybová chyba

    Ujistěte se, že vaše databázové řetězce jsou správně nastaveny pro N5. Viz Database Setup pro pokyny.

  • MySql.Data.MySqlClient.MySqlException: Table '<database>.boothistories' neexistuje.

    Pokud se pokusíte spustit server poprvé a přijmout tuto chybu, jednoduše spusťte databázi a zkuste to znovu.

FiveM License Key

  • GlobalError: Could not authenticate server license key.

    This error means you do not have a valid FiveM license key in server.cfg, make sure the file contains a valid sv_licensekey line.

Chyby klientů

  • System.Runtime.Serialization.dll is not a platform image (even though the dir matches).

    Tuto chybu lze bezpečně ignorovat.
