Finn plugins

Fullstendig administrert

NFive er 100% skrevet i C# .NET og kjører som en enkelt server ressurs.

Modulær utforming

NFive sitt plugin system er modulært og designet for komplekse avhengigheter, med full semantisk versjonering.

Software Development Kit

NFive has a full featured SDK which simplifies and eases plugin and mod development.

100% åpen kildekode

Hver NFive komponenten er helt offentlig, åpen kildekode og gratis.

Prøv det ut

Sett opp din egen FiveM og NFive server på sekunder, prøv det nå!

Moderne teknologi

NFive er bygget ved hjelp av moderne programmering praksiser og teknologier. Utnytt kraften til CI/CD, unit testing, I18N, semantisk versjonskontroll og mer i koden.

Automatiserte verktøy

Kom i gang med å lage nye plugins på noen få sekunder med innebygde verktøy. Generer en plugin med en kommando eller automatisk generer en database migrasjon; la verktøyene lage grunnmuren for deg.
